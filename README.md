# Safe Report Analyzer

A simple Node.js utility for analyzing structured report sequences based on validation rules and directional consistency checks.

## Features
- Validates sequential numeric reports
- Detects increasing/decreasing patterns
- Ensures allowed difference thresholds
- Processes multiple report inputs

## Tech Stack
- JavaScript / Node.js

## Example
Input:
7 6 4 2 1

Output:
Safe Report: true
